# Scan Docker Image

This script purpose is to scan Docker images for vulnerabilities.

## Get a token:

https://microscanner.aquasec.com/signup

## Usage:

```SCANNER_TOKEN=<TOKEN> SCANNER_IMAGE=jboss/keycloak:6.0.1 ./docker-scan.sh --silent```
